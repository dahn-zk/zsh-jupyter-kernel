## installation

[PyPi](https://pypi.org/project/zsh-jupyter-kernel/):

Pip
```sh
python -m pip install zsh_jupyter_kernel
```
   
Pipenv
```sh
pipenv install zsh_jupyter_kernel
```

### install kernel file

see the help command for details
```sh
python -m zsh_jupyter_kernel.install --help
```
